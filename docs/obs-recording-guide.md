# OBS Studio 撮影ガイド — ATTRACT チュートリアル動画

## 事前準備

### 1. 画面配置（2モニター推奨）

**メインモニター（録画対象）**: Chrome で ATTRACT を全画面表示
**サブモニター（または別ウィンドウ）**: 撮影コンソール（tutorial-player.html）

> 1モニターの場合: Chromeを左半分、コンソールを右半分に配置し、
> 録画時は Chrome ウィンドウのみをキャプチャ

### 2. OBS Studio 初回設定

1. OBS Studio を起動（`C:\Program Files\obs-studio\bin\64bit\obs64.exe`）
2. 自動設定ウィザードが出たら「録画のために最適化」→ 次へ → 完了

### 3. シーン＆ソース設定

#### ソース追加
1. 「ソース」パネルの「+」ボタン
2. 「ウィンドウキャプチャ」を選択 → 名前「Chrome」
3. ウィンドウ: `[chrome.exe]: ATTRACT` を選択
4. 「カーソルをキャプチャ」✅ オン

#### 音声設定
1. 「ソース」の「+」→「メディアソース」→ 名前「ナレーション」
2. ローカルファイル: OFF（後で切り替えるため、最初は空でOK）
3. または、コンソール側で音声を再生し「デスクトップ音声」で録音

**推奨方法**: デスクトップ音声で録音（コンソールの再生音をそのままキャプチャ）

#### 録画設定
1. 設定 → 出力 → 録画
   - 録画パス: `C:\Users\K-USER01027\Videos\attract-tutorial`
   - 録画フォーマット: **mp4**
   - エンコーダ: **x264** or ハードウェア（NVENC等）
   - レート制御: CBR
   - ビットレート: **6000 Kbps**（高品質）
2. 設定 → 映像
   - 基本解像度: **1920×1080**
   - 出力解像度: **1920×1080**
   - FPS: **30**

### 4. 録画前チェックリスト

- [ ] Chrome: ブックマークバー非表示（Ctrl+Shift+B）
- [ ] Chrome: タブ1つだけ（不要タブを閉じる）
- [ ] Chrome: 拡張機能アイコン非表示
- [ ] デスクトップ通知OFF（集中モード/おやすみモード）
- [ ] マウスカーソルを大きめに設定（設定 → アクセシビリティ → カーソルサイズ）
- [ ] ATTRACT: https://attract-prototype.vercel.app/ を開いておく
- [ ] 撮影コンソール: tutorial-player.html をブラウザで開いておく

## 撮影手順

### 方法A: シーン別に分割録画（推奨）

各シーンごとに録画開始→停止を繰り返す方法。ミスしたシーンだけ撮り直しOK。

1. OBSで「録画開始」ボタン
2. 撮影コンソールでシーン0の「▶ 再生」をクリック
3. ナレーション音声が流れる → 音声に合わせて画面操作
4. 音声終了 → 2〜3秒待ってOBSで「録画停止」
5. 次のシーンで同様に繰り返し
6. 最後に全mp4ファイルを動画編集ソフトで結合

### 方法B: 一気撮り

1. OBSで「録画開始」
2. 撮影コンソールでシーン0から順に再生（→キーで次へ進む）
3. 全シーン終了後「録画停止」
4. 1本のmp4ファイルが完成

### 操作のコツ

- **マウスはゆっくり動かす**（急な動きは画面酔いの原因）
- **クリック前に1秒止める**（「ここをクリックします」の間）
- **スクロールはセクション単位**で止めながら
- **ナレーション音声のペースに合わせる**（急がない）
- ミスしたら慌てず停止→そのシーンだけ撮り直し
